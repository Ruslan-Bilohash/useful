# 🚀 МОЩНИЙ СБОРЩИК EMAIL 

Потужний PHP-скрипт для автоматичного збору email-адрес компаній та рекрутерів.

### ✨ Основні можливості

- Автоматичний парсинг усіх сторінок пошуку
- Перехід на кожну вакансію та витягування email-адрес з тексту сторінки
- Красивий кольоровий вивід у консолі з прогресом і емодзі
- Зберігає результати **одразу** у два формати:
  - `NAV_IT_Emails_YYYY-MM-DD_HH-MM-SS.txt` — читабельна таблиця
  - `NAV_IT_Emails_YYYY-MM-DD_HH-MM-SS.json` — структуровані дані
- Захист від дублювання на одній сторінці
- Комфортна затримка між запитами (≈1.2 секунди)
- Підтримка User-Agent + Accept-Language (норвезька)
- Повна статистика та красивий фінальний звіт

### Що вміє цей інструмент

Цей скрипт — частина мого великого набору інструментів автоматизації.  
Я спеціалізуюсь на:

- Масовому зборі інформації та лідів
- Написанні скриптів парсингу та автоматизації
- Зборі email-баз з державних та комерційних сайтів
- Безпечній розсилці без ризику потрапляння в спам
- Створенні власних CRM-систем для автоматизації продажів і рекрутингу
- Розробці потужних інструментів пошуку IT-компаній та маркетингових агенцій
